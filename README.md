## Skill Stack

### Mandatory (Fresher Roles)
- [ ] Java — OOP, Collections, Exception Handling
- [ ] Selenium WebDriver — locators, waits, interactions
- [ ] TestNG — annotations, data providers, parallel config
- [ ] Page Object Model — clean architecture
- [ ] Maven — dependency management, pom.xml
- [ ] Git/GitHub — version control

### Mid-Level Requirements
- [ ] Selenium Grid — parallel execution
- [ ] Jenkins / GitHub Actions — CI/CD
- [ ] Apache POI — Excel data-driven testing
- [ ] Allure Reports — rich HTML reporting
- [ ] SQL basics — database validation
- [ ] Cucumber BDD — Gherkin test scenarios

### Senior-Level
- [ ] Chrome DevTools Protocol — Selenium 4
- [ ] Framework architecture design from scratch
- [ ] Docker + Selenium Grid setup
- [ ] REST API testing (Rest-Assured)
- [ ] Appium basics — mobile testing extension

---

## Portfolio Requirements

Minimum 3 deployable projects:
- [ ] E-commerce Regression Suite — POM, TestNG, Allure
- [ ] Data-Driven Login Tests — Apache POI, 50+ scenarios
- [ ] Cross-Browser Suite — Grid parallel Chrome + Firefox
- [ ] BDD Framework — Cucumber, Gherkin, step definitions

All on GitHub with README, pom.xml, testng.xml, and sample reports.
